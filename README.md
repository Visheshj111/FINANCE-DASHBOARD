# Finance Dashboard

A responsive finance dashboard built with Next.js 15 and Tailwind CSS. Tracks transactions, visualizes balance trends, and supports role-based access all in a clean grayscale UI with dark mode.

## What's inside

Filter your balance trend chart by date range directly from the chart view. Add transactions from anywhere using the global input in the nav bar. Switch between Viewer and Admin roles only Admins can add or delete transactions. Charts are powered by Recharts (Area and Pie), and dark mode is handled via Next.js Themes.

## Stack

Next.js 15 (App Router), React 19, TypeScript, Tailwind CSS v4, Recharts

## Getting started
```bash
npm install
npm run dev
```
