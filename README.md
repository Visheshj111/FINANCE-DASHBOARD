# Finance Dashboard

A clean, monochromatic, and fully responsive frontend finance dashboard built using Next.js 15, React, and Tailwind CSS.

## Features

- **Monochromatic Aesthetic**: Strict black and white styling, explicitly designed without emojis or excessive colors (except meaningful red/green indicators for transaction amounts).
- **Date Filtering**: Chart data can be isolated by specific dates directly alongside the Balance Trend chart.
- **Transactions Management**: Complete global module allowing users to input transactions directly from the navigation bar on any page.
- **Dynamic Role State**: Simulated Viewer and Admin roles. Only the Admin role can add or delete transactions.
- **Visual Analytics**: Interactive data visualization powered by Recharts (Area and Pie charts), seamlessly adapting to the grayscale aesthetic and dark mode parameters.
- **Dark Mode**: Integrated robust full-layout toggling utilizing Next.js Themes.

## Tech Stack
- Next.js 15 (App Router)
- React 19
- Tailwind CSS v4
- Recharts (Data Visualization)
- TypeScript

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the outcome.

Enjoy managing your finances smoothly!
